## ═══════════════════════════════════════════════════════════════════════════
## HOW TO ADD VARIABLES TO RENDER
## ═══════════════════════════════════════════════════════════════════════════

# 1. Go to: https://dashboard.render.com
# 2. Click on service (backend or frontend)
# 3. Click "Environment" tab
# 4. Click "Add Environment Variable"
# 5. For each variable:
#    - Key: [variable name]
#    - Value: [variable value]
# 6. Click "Save Changes"
# 7. Service auto-redeploys with new variables

## ═══════════════════════════════════════════════════════════════════════════
## IMPORTANT NOTES
## ═══════════════════════════════════════════════════════════════════════════

🔐 SECURITY:
- Generate a NEW JWT_SECRET: https://generate-random.org/string-generator
- Never share environment variable values
- Keep API keys private
- Use app-specific passwords (not main account password)

📝 REQUIRED CHANGES:
- JWT_SECRET (must be 32+ characters)
- BACKEND_URL (update after deployment)
- FRONTEND_URL (update after deployment)
- Database credentials (from Render)

✅ OPTIONAL (only if using):
- Payment gateway keys (Razorpay, Stripe)
- Email credentials (Gmail, Brevo)
- API keys (Anthropic, etc.)

🔄 AFTER ADDING VARIABLES:
- Render auto-redeploys services
- Takes 1-2 minutes to restart
- New variables take effect immediately
